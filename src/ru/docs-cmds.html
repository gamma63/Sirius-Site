Документация к командам

Команды для получения информации

help : выводит вспомгательную информацию

ver : выводит версию ОС Сириус

pcinf : выводит информацию о компьютере (ЦПУ и ОЗУ)

Команды файловой системы

fvfs touch 0:\f.txt : создаёт файл

fvfs mkdir 0:\System32 : создаёт директорию

fvfs rm 0:\f.txt : удаляет файл

fvfs rm -r 0:\System32 : удаляет директорию

fvfs write 0:\f.txt : небольшой текстовый редактор

fvfs cat 0:\f.txt : читает файл

Команды для системы питания

reboot : перезагружает компьютер

poweroff : выключает компьютер
