Commands documentation

Info Commands

help : printing help information

ver : displays current version of SiriusOS

pcinf : shows computer information (CPU and RAM)

File System Commands

fvfs touch 0:\f.txt : creates a file

fvfs mkdir 0:\System32 : creates a directory

fvfs rm 0:\f.txt : removes a file

fvfs rm -r 0:\System32 : removes a directory

fvfs write 0:\f.txt : small text editor

fvfs cat 0:\f.txt : reads a file

Power Management commands

reboot : reboots computer

poweroff : shuts down the computer
